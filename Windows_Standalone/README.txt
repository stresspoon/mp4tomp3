MP4 to MP3 Converter - Windows Standalone EXE
==============================================

📌 이 폴더의 용도:
Windows에서 Python 설치 없이 실행 가능한 
단독 실행 파일(.exe)을 만들기 위한 폴더입니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔨 EXE 파일 만들기 (Windows에서 실행):

1. Windows PC로 이 폴더 전체 복사
2. build.bat 더블클릭
3. dist 폴더에 MP4toMP3.exe 생성 (약 90-100MB)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 생성된 EXE 파일의 특징:

• Python 설치 불필요
• ffmpeg 내장
• 단일 파일로 배포 가능
• Windows 7/8/10/11 호환
• 바로 실행 가능

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📦 포함된 파일:

• converter_universal.py - 메인 프로그램 소스
• ffmpeg.exe - 변환 엔진 (83MB)
• build.bat - 빌드 스크립트
• README.txt - 이 파일

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 주의사항:

• 빌드는 Windows에서만 가능
• 빌드 시 Python 필요 (빌드용)
• 빌드된 exe는 Python 없이 실행
• 첫 실행 시 Windows Defender 경고 가능 (정상)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 배포 방법:

1. dist\MP4toMP3.exe 파일 하나만 전달
2. 받는 사람이 더블클릭으로 실행
3. 추가 설치 없음!
